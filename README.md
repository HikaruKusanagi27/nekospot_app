# 猫スナップ

### コンセプト
街角で出会った可愛い猫たちをシェアするアプリ。
日常の忙しさの中で、ふと目にした猫との偶然の出会いを大切な思い出として残す。
「猫スナップ」では、あなたが見つけた猫の写真を簡単に投稿でき、場所や状況も記録できます。
全国の猫好きが投稿した素敵な猫たちの写真を眺めて、日々の癒しをもらいましょう。
